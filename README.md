Repositorio para elaborar la tarea 1 de la asignatura Programación de Inteligencia Artificial del Curso de Especialización en IA y Big Data en CIDEAD

La tarea consiste, de manera resumida, en dos apartados:

1. Crear una cuenta en GitHub y crear un sencillo repositorio con dos archivos, este README.md y un archivo HolaMundo.py
2. Conectar con la comunidad de Inteligencia Artificial a través de redes sociales
